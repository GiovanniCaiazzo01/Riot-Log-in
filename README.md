# Riot Login UI

A clean and responsive Riot / Valorant-inspired login page built with HTML, CSS, and JavaScript.
<img width="2281" height="1400" alt="image" src="https://github.com/user-attachments/assets/196cb881-1dcf-4a85-bf52-71529401d1ea" />
## Demo
Check out the live demo [here](https://giovannicaiazzo01.github.io/Riot-Log-in/)!

## Features

- Responsive split-screen layout
- Rotating agent backgrounds
- Floating label inputs
- Password show / hide toggle
- Demo login validation
- Modern UI

## Technologies

- HTML
- CSS
- JavaScript

## Purpose

This project is a front-end login concept designed to recreate a more polished game-style authentication experience.
